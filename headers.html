The Weight of Metadata

Not all headers are created equal. While General headers appear in almost every message, Entity headers only appear when data (like a file or HTML body) is present. Security headers are crucial but often optional (unfortunately).

Key Insight

The Host header is mandatory in HTTP/1.1 requests. Without it, the server doesn't know which website you want to visit on that IP address.

Security Headers: The Shield

Security headers instruct the browser to enforce safety measures. Without them, users are vulnerable to XSS, Clickjacking, and Protocol Downgrades.

  • Strict-Transport-Security HSTS

    Forces the browser to ONLY use HTTPS. Stops protocol downgrade attacks.

  • Content-Security-Policy CSP

    Restricts where scripts can be loaded from. The best defense against XSS.
