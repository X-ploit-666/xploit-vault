Decodng the Output

The spider stores its findings in a results.json file. This "digital filing cabinet" organizes data into specific keys. Understanding this structure is crucial for post-scan analysis.

Network Map (Links & Files) URLs of internal pages, external PDFs, and documents.
Code Artifacts (JS & Comments) JavaScript files and developer HTML comments (often containing clues).
Contact Points (Emails & Forms) Email addresses for phishing vectors and form fields for injection testing.
