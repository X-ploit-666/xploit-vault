What is WHOIS?

WHOIS is a widely used query and response protocol designed to access databases storing details about internet resources. It functions as a **massive digital directory**, allowing users to identify the owners of domains, IP address blocks, and autonomous systems.

Evolution of the System

From ARPANET origins to GDPR compliance.

1970s: The Origin

Created by **Elizabeth Feinler** at SRI. Designed to track users and hostnames on the ARPANET (Pre-Internet).

1982: Standardization

Formalized in **RFC 812**. This established the structured rules needed for the system to scale with the growing internet.

1998: ICANN & RIRs

Formation of **ICANN** for global management. Shift to **Regional Internet Registries (RIRs)** to decentralize the database.

2018+: GDPR & RDAP

Privacy concerns led to **GDPR**, redacting personal info. **RDAP** is now emerging as the secure, privacy-conscious successor.

Reconnaissance Value

For penetration testers, WHOIS is a treasure trove. It provides critical insights into the target's digital footprint, helping to identify vulnerabilities before an attack occurs.

Key Use Cases

Finding Targets (Social Engineering)

Identifying key personnel names and emails to craft targeted phishing campaigns or social engineering attacks.

Infrastructure Mapping

Discovering Name Servers and IP blocks to find potential entry points, misconfigurations, or "back doors".

Historical Analysis

Using tools to view past WHOIS records, tracking how a company's ownership or infrastructure has changed over time.
