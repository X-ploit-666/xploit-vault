APIs: The Bridge

An API (Application Programming Interface) is the messenger. It takes a request from the user (Front End) and tells the system (Back End) what to do.

1 Query Parameters

The simplest way to pass data. Think of it as writing instructions on the outside of an envelope vs. inside.

GET REQUEST // Visible in URL
search.php?item=apples
POST REQUEST // Hidden in Body
POST /search.php HTTP/1.1
...headers...

item=apples

2 REST Methods

In the modern REST standard, we use specific "Verbs" to tell the server what action to take on a resource.
